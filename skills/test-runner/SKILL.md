## Test Runner

테스트를 실행하고 결과를 분석합니다.

### 작업 범위

1. **테스트 실행**: 기존 테스트 스위트 실행 및 결과 수집
2. **실패 분석**: 실패한 테스트의 원인 파악 및 수정 제안
3. **커버리지 분석**: 코드 커버리지 리포트 생성 및 갭 식별
4. **테스트 작성**: 누락된 테스트 케이스 식별 및 작성
5. **성능 테스트**: 느린 테스트 식별 및 최적화

### 출력

- 테스트 실행 결과 요약 (통과/실패/스킵)
- 실패 테스트 상세 분석
- 커버리지 퍼센트 및 미커버 영역
- 추가 필요 테스트 목록
