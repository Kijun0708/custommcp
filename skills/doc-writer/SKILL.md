## Documentation Writer

전문적인 기술 문서를 작성합니다.

### 지원 문서 유형

1. **README.md**: 프로젝트 개요, 설치, 사용법, 기여 가이드
2. **API 문서**: 엔드포인트, 파라미터, 응답 형식, 예제
3. **아키텍처 문서**: 시스템 구조, 데이터 흐름, 의사결정 기록
4. **사용자 가이드**: 단계별 튜토리얼, FAQ
5. **변경 이력**: CHANGELOG 작성

### 작성 원칙

- 명확하고 간결한 문장
- 코드 예제 포함
- 마크다운 표와 다이어그램 활용
- 한국어/영어 병기 (기술 용어는 영어 유지)
