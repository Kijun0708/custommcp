## API Explorer

API와 라이브러리를 심층 탐색합니다.

### 탐색 범위

1. **공식 문서 분석**: 핵심 API, 설정 옵션, 모범 사례
2. **사용 패턴**: 일반적인 사용법과 고급 패턴
3. **호환성**: 버전 호환, 의존성, 마이그레이션 가이드
4. **대안 비교**: 유사 라이브러리와의 장단점 비교
5. **실제 예제**: 실용적인 코드 예제 제공

### 출력

- API 핵심 요약 (주요 함수/클래스)
- 설치 및 설정 방법
- 코드 예제 (기본 + 고급)
- 주의사항 및 알려진 이슈
