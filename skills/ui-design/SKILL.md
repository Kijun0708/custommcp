## UI Design

UI/UX 관점에서 컴포넌트를 설계하고 구현합니다.

### 설계 원칙

1. **접근성 (a11y)**: WCAG 2.1 AA 준수, 키보드 내비게이션, 스크린 리더
2. **반응형**: 모바일, 태블릿, 데스크톱 대응
3. **성능**: 렌더링 최적화, 레이지 로딩, 번들 크기
4. **일관성**: 디자인 시스템 및 토큰 활용
5. **인터랙션**: 애니메이션, 트랜지션, 피드백

### 출력

- 컴포넌트 구조 (props, state, 이벤트)
- 스타일링 코드 (CSS/Tailwind/styled-components)
- 접근성 속성 (aria-*, role)
- 반응형 브레이크포인트
